# Update Extensions

- `$add(value)`: increment numbers or add values to sets depending on field type
- `$sum(left, right)`: numeric addition expression
- `$subtract(left, right)`: numeric subtraction expression
- `$append(list)`: append list items
- `$prepend(list)`: prepend list items
- `$delete(set)`: remove set members
- `$remove()`: remove an optional attribute
- `$get(path, fallback?)`: reference another attribute path in the same update
